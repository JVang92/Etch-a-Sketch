# Etch-a-Sketch

A web based sketchpad built as part of The Odin Project. This project demonstrates DOM manipulation, event listeners, and dynamic grid generation using Flexbox.

[View Live Demo](https://jvang92.github.io/Etch-a-Sketch/)

## Features
- **Dynamic Grid:** Users can choose a custom grid size (up to 100x100).
- **Random Color Mode:** Toggle between a standard grey color and a randomized RGB "Rainbow" mode.
- **Responsive Design:** Built using **Flexbox** to ensure the grid stays within a fixed 600px container regardless of the number of squares.

## Built With
- **HTML5**
- **CSS3** (Flexbox)
- **JavaScript** 

## Lessons Learned
- **DOM Manipulation:** How to create, append, and clear hundreds of elements efficiently using JavaScript loops.
- **Flexbox Logic:** Calculating dynamic widths and heights in JavaScript to ensure items wrap perfectly within a container.
- **Event Listeners:** Using `pointerover` to create a smooth drawing experience.
